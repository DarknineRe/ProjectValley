
  # ระบบจัดการสินค้าประเภทผลผลิตทางการเกษตร

This repository contains the frontend code for the Agricultural Product Management system. The design originated in Figma but the codebase is self‑contained.

For deployment you must provide the backend URL to the frontend via an env var, e.g.:
```
VITE_API_URL=https://farmvalley.onrender.com
```

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.
