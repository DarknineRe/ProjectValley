
  # ระบบจัดการสินค้าประเภทผลผลิตทางการเกษตร

This repository contains the frontend code for the Agricultural Product Management system. The design originated in Figma but the codebase is self‑contained.

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.
